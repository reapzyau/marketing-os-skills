# Post Templates

Fill-in-the-blank LinkedIn templates for each framework. Character count guidance included.

---

## Template 1: Educational List (800-1300 chars)

```
[Hook — under 125 chars, before the fold]

[1-2 sentences of context — why this matters to the reader]

1. [Item]
[1-2 sentences explaining why/how]

2. [Item]
[1-2 sentences explaining why/how]

3. [Item]
[1-2 sentences explaining why/how]

4. [Item]
[1-2 sentences explaining why/how]

5. [Item]
[1-2 sentences explaining why/how]

[Takeaway — one sentence tying it together]

[Engagement prompt — specific question, not "thoughts?"]
```

---

## Template 2: Story + Lesson (800-1300 chars)

```
[Hook — set the scene or state the surprising outcome, <125 chars]

[The situation — 2-3 sentences, make it relatable]

[The turning point — what changed, 1-2 sentences]

[The lesson — stated clearly in one sentence]

[Application — how the reader can use this, 1-2 sentences]

[Engagement prompt — "Have you experienced something similar?"]
```

## Template 3: Old Way vs New Way (600-900 chars)

```
How to [outcome]:

Old way:
- [Outdated approach]
- [Outdated approach]
- [Outdated approach]
- [Outdated approach]

New way:
- [Modern approach]
- [Modern approach]
- [Modern approach]
- [Modern approach]

[Why the shift matters — 1-2 sentences]

[One-word or one-line principle]
```

## Template 4: Two Types of People (600-900 chars)

```
There are two types of [people]:

Type 1: [Description — 2-3 sentences about behavior and result]

Type 2: [Description — 2-3 sentences about behavior and result]

The difference? [One insight]

[Which one are you? / Which one do you want to be?]
```

## Template 5: Biggest Mistake (800-1300 chars)

```
The biggest mistake I made as a [role]?

[State the mistake — one sentence]

[Context — why it happened, what I was thinking, 2-3 sentences]

Instead, [do this]:

1. [Fix 1]
2. [Fix 2]
3. [Fix 3]
4. [Fix 4]
5. [Fix 5]

[One line of additional context]

[The outcome — what changed when I fixed it]
```

## Template 6: Framework Share (800-1300 chars)

```
[Hook — name the framework or promise the outcome, <125 chars]

Here's my [name] framework:

Step 1: [Verb]
[2-3 sentences explaining what and why]

Step 2: [Verb]
[2-3 sentences explaining what and why]

Step 3: [Verb]
[2-3 sentences explaining what and why]

[Result — what happens when you follow this, with numbers if possible]

[Save this for later / Which step do you struggle with?]
```

---

## Template 7: Hot Take (600-1000 chars)

```
[The hot take — bold, one sentence, <125 chars]

Everyone says [conventional wisdom].

But here's what I've seen:
[Your evidence — specific story, data point, or logical argument, 3-5 sentences]

The truth: [reframe in one sentence]

[Am I wrong? / Agree or disagree? / What's your take?]
```

## Template 8: Do These X Things (800-1300 chars)

```
If you're [specific identity/situation]:

Stop.

You must [do/have/know] these [N] things:

1. [Thing]
[Brief context — why it matters]

2. [Thing]
[Brief context]

3. [Thing]
[Brief context]

[Formula: X + Y = Z outcome]

[Engagement prompt]
```

## Template 9: Research/Data Share (500-1500 chars)

```
[Hook — "I analysed [number] [things] to [find] [insight].", <125 chars]

[Setup — why this matters, 1-2 sentences]

The data reveals:
1. [Finding with specific number] — [implication]
2. [Finding with specific number] — [implication]
3. [Finding with specific number] — [implication]

[What this means for the reader — one sentence]

[Contrarian reframe or "so what"]

Full analysis in the comments.
```

## Character Count Reference

| Element | Limit |
|---------|-------|
| Maximum post length | 3,000 chars |
| Optimal post length | 500-1,500 chars |
| Hook (before "see more" fold) | 125 chars |
