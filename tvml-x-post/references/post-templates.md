# Post Templates

Fill-in-the-blank templates for each framework, with X-specific variants. Character count guidance included.

---

## Template 1: Educational List

### X Single Version (<280 chars)

```
[Hook + list in one tweet]

[N] [things] for [outcome]:

1. [Item]
2. [Item]
3. [Item]

[One-line takeaway]
```

### X Thread Version (5-8 tweets)

```
Tweet 1 (Hook):
[N] [things] to [outcome]

A thread:

Tweet 2 (Promise):
By the end you'll know:
- [Outcome 1]
- [Outcome 2]
- [Outcome 3]

Tweets 3-N (One item per tweet):
[Number]. [Item]

[2-3 lines of context/explanation]

Final Tweet (CTA):
That's all [N].

[Restate the value]

Follow me @[handle] for more on [topic].
```

## Character Count Reference

| Platform | Max | Optimal | Hook Limit |
|----------|-----|---------|------------|
| X single | 280 chars | <200 chars (for QT-ability) | N/A (whole tweet is hook) |
| X thread tweet | 280 chars/tweet | 200-250 chars/tweet | Tweet 1 = hook |
